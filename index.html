タスク登録の自動化

  • 登録

    • asana、zapierにアクセスし、右上の無料で始める、Sign upからGoogleアカウントで登録する。

    asanaはモバイルアプリからでも登録できます

    必ず同じGoogleアカウントで登録してください。

  • 作成
    1. asanaにアクセスし、新規プロジェクトからプロジェクト名を入力し作成する。
    2. aにアクセスし、下のTry this Zapを押す。
    3. New Email Matching SearchにあるSign inをタップし、今回使っているアカウントのGmailでサインインする。
    4. Configureをタップし、Search keywordsに

      下記の授業資料が登録または更新されました。

      と入力する。

    5. Testから、Test triggerをタップし、右のような画面になったら下のContinue with selected recordをタップする。
    6. Create Taskをタップし、AccountのSign inから先ほど作ったasanaのアカウントにサインインする。
    7. Configureをタップし、以下の通りに選択する。

      Projects:先ほど作ったプロジェクト

      Section:(セクションなし)

      Due date type:Date only

      Due at:今回はTomollowにしている(締め切りの曜日を指定)

      Name:横の＋を押し、Body Plainを選択

      Is the description rich text?:False

      Mark Task as complete?:False

      Mark Task as liked?:False

      Assignee:自身のアカウント

      Followers:自身のアカウント

    8. TestからTest stepを選択し、右のような画面になったらPublishを選択
